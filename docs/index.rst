.. _sentry-cli:

Command Line Interface
======================

For certain actions you can use the ``sentry-cli`` command line
executable.  It can connect to the Sentry API and manage some data for
your projects.  It's primarily used for managing debug information files
for iOS, Android as well as release and sourcemaps management for other
platforms.
